# 如何拿到产出？
>本科生想发论文or专利，除了大创，百步梯，攀登之类，最好也是最容易的方式就是SRP，如何在保研更加浓墨淡彩一笔或是出国加一把削铁如泥的利器？往下看
### 方法一：报名本院热门的项目
- 首先，每个老师风格不一，但是火的项目一定有火的理由
- 其次，你可以点进详情，发现产出很多那种，跟住，到最后大概率会有产出
- 但注意，计院不仅竞争大容易被刷，进去了你也要知道其他人也是抱着同样想法的，u ought to work hard if u wanna any outcome
### 方法二：报名其他院的项目（自己简历一定要足够优秀）
- 首先，本院大部分都是LLM等大模型项目，我们可以着手于软件学院，那里竞争小，而且产出也有各种开发方向，但一定要看好研究过程，那里可以看到有多少产出。
- 其次，可以看下自动化学院、工商管理、旅游甚至法学，这些学院就报最热门而且代码相关的，老师选你，就一定有代码相关需求，你要做好孤军奋战的打算，被甲方包围的感觉什么的不要啊，不过能产出就是了...
## 无论哪种办法，一定要找到你心仪项目的导师邮箱，给他发邮件介绍自己，别被选的人数吓到了，能发邮件已经是区分大多数人的一个分水岭了
