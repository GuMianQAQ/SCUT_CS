# 如何混到创新分？
>华南理工的毕业创新分要求总是难到众多好汉，大多数玩家其实并不需要相关技能，但是苦于没有创新分，又不想收高额讲座票，既要又要怎么办？往下看
### 方法一：报名本院较水的项目
- 首先，每个老师风格不一，可以优先寻找低报名量项目，你可以减少竞争，同时这种水的概率比较大
- 其次，你可以点进详情，发现产出可能只有一篇论文的那种，放心大胆跟，到最后大概率是什么产出没有的超水项目
### 方法二：报名其他院的项目
- 如果英语好，直接无脑外国语学院，可以参考我的srp项目，水的要死，到最后我看不下去不水了，发现领队在毕业，老师在另一个半球度假| 为了避免不必要的麻烦，我把老师名字在相关资料删掉了
- 既要又要英语还不好？工商管理，旅游，这些你就选吧，但一定注意，这些学院也不是都水，老师也大概率选择的本院的，选你你要做好最坏的打算，给他们搞出一个平台之类的，不过也算是莫名产出了...
